# icons/

ICO artwork for the Deep Analysis Agent tray service.

## Files (generated, not checked in)

```
W.ico       # White mana pip — uploading state
U.ico       # Blue mana pip — uploading state
B.ico       # Black mana pip — uploading state
R.ico       # Red mana pip — uploading state
G.ico       # Green mana pip — uploading state
C.ico       # Colorless pip — idle state
identity.ico  # 5-wedge WUBRG identity pie — app icon / installer
```

ICO files are generated by `icons/generate_icons.py` (to be written in Phase 3) and are listed in `.gitignore`. They are not checked in as binary assets.

## Tray behavior

- **Idle:** C.ico (colorless pip)
- **Uploading:** cycles W → U → B → R → G, advancing on each upload tick
- **Error / disconnected:** TBD (likely red indicator)

## Design notes

This design comes from manalog v0.3.7/v0.3.8, where the WUBRG cycling icon was validated as visually distinctive and recognizable to MTG players. The identity-pie icon (all five colors in wedge segments) survives the Deep Analysis rebrand unchanged.

Read `workspaces/manalog/agent/generate_icons.py` for the generation approach. Do not copy the code wholesale — write fresh for this repo, but the visual design and file structure are keepers.
